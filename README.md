Wat is dit project? Voor deze opdracht heb een app gemaakt voor euromoon trein. Het programma is geschreven in Java. Ik heb geprobeerd om een zoals een echte app te laten lijken dus je start als een nieuwe gebruiker, je maakt een account aan, en daarna kan je inloggen om tickets te kopen.

Wat kan je doen in de app?

Account maken: Je kan je eigen gegevens invullen en een wachtwoord kiezen.
Inloggen: Alleen met de juiste gegevnes kom je in het menu.
Trein station kiezen: Je kan kiezen tussen twee bestemmingen Londen of Parijs en zien hoeveel plaatsen er nog vrij zijn.
Tickets: Je kan een ticket kopen en die daarna nog controleren of annuleren.
Geraadpleegde bronnen Ik heb voor dit project geen volledige codeblokken gekopieerd ik heb wel aan gemini voor soms verbetering gevraagd en lesmateriaal. De logica van de app heb ik zelf geschreven adhv mijn user stories Ik heb ook voor de volgende onderdelen specifieke tutorials geraadpleegd om de sommige Java-concepten beter te begrijpen:

ArrayLists (Dynamische lijsten): Ik heb deze video gebruikt om te leren hoe ik passagiers en tickets in een lijst kan opslaan die groter kan worden. Java ArrayList Tutorial - Bro Code

FileWriter (Boardinglijst exporteren): Om de boardinglijst naar een tekstbestand te kunnen schrijven, heb ik de basis van de FileWriter klasse hier bekeken: How to write to a file in Java - Alex Lee

Inheritance (Overerving): Voor het maken van de structuur tussen Persoon, Passagier en Personeel heb ik deze uitleg over overerving gebruikt: Java Inheritance Tutorial

Scanner & Menu Loops: Om een menu te maken dat pas stopt als de gebruiker dat wil, heb ik me gebaseerd op de technieken in deze video: Java Scanner en While Loops

Hoe de app gebruiken:

Open de EuromoonApp.java in een IDE (zoals IntelliJ).
1.Start het programma.
2.Kies in het hoofdmenu voor 'Registreren' om een account aan te maken.
3.Log in met je nieuwe gegevens om door te gaan naar het boekingssysteem.
